# Sample Documents for SecureRAG Demo

This folder contains sample documents for testing the SecureRAG system.

## Files

- `refund_policy.txt` - Company refund policy document
- `merchant_guide.txt` - Merchant onboarding guide
- `internal_secrets.txt` - Sensitive document (for testing tenant isolation)

## Usage

These documents are automatically loaded when the API starts.
They demonstrate:
- Normal document retrieval
- Tenant isolation (company_a vs company_b)
- Citation generation
