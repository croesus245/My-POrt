Components

  • Streaming feature pipeline: Kafka â†’ Redis with velocity aggregates and entity embeddings
  • Scoring service: FastAPI with XGBoost model, returns probability + decision
  • Label reconciler: Joins confirmed fraud labels (30+ day lag) with predictions
  • Drift monitor: PSI on predictions, KS test on features, fraud rate tracking
  • Retrain trigger: Automated pipeline when drift exceeds thresholds

What I Broke

What Happened

  • Alert fired, on-call paged
  • Slice analysis identified affected merchant category
  • Retraining triggered with augmented data
  • New model deployed within 48 hours
