Security Layers

  • Input sanitizer: Detect/neutralize injection patterns before retrieval
  • Permission filter: Only retrieve documents user has access to
  • Tool sandbox: Tools have explicit capabilities; no shell access
  • Output validator: Check for PII, injection echoes, hallucination markers
